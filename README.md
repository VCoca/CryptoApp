# Encrypted File Transfer Application

Desktop application that encrypts and transfers files between users.

## Features

- File encryption before transfer
- File decryption after transfer
- File watcher that monitors a directory for new files
- Secure file transmission
- Simple desktop interface

## Technologies

- C#
- WPF
- Networking
- Encryption

## What I learned

This project helped me understand file encryption techniques,
network communication between applications, and building
desktop interfaces using WPF.
